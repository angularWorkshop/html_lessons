# Exercise: Responsive Image Cards

## Goal
Create responsive image cards with aspect-ratio, object-fit, and a grid that adapts from 1 to 3 columns.

## Checklist
- [ ] `aspect-ratio: 3/2` on card images
- [ ] `object-fit: cover` for image cropping
- [ ] Grid: 1 column mobile, 3 columns desktop
- [ ] `max-width: 100%` on images
