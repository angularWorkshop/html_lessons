# Exercise: Holy Grail Layout

## Goal
Build the classic "Holy Grail" layout using Flexbox: header, two sidebars, main content, footer.

## Checklist
- [ ] Body: flex column, min-height 100vh
- [ ] Middle section: flex row
- [ ] Content: flex 1 (fills remaining space)
- [ ] Sidebars: flex 0 0 200px (fixed width)
- [ ] Footer pushed to bottom
